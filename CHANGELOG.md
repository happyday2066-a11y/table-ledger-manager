# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0 - 2026-03-13

- First public release of `table-ledger-manager`.
- Publish a usable CLI workflow for importing, querying, updating, exporting, and rebuilding embeddings for structured ledgers.
- Publish the FastAPI Web UI for local import, query, and export flows.
- Add repository metadata required for public OSS maintenance: `LICENSE`, CI workflow, contributing guide, and privacy guidance.
- Replace public examples and tests with synthetic sample data and document that runtime data should stay outside the repository.
